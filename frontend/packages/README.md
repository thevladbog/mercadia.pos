# Frontend Packages

Shared frontend packages and generated frontend API clients will live here.

Expected packages:

- `api-clients` — Orval-generated HTTP clients
- `ui` — `@mercadia/ui` Radix components and theme system (see [`ui-components.md`](../../docs/development/ui-components.md))
- `design-tokens` — reserved for extracted token artifacts if needed
- `testing` — shared frontend test utilities

Generated API clients must come from OpenAPI through Orval and must not be edited manually.
