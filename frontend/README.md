# Frontend

This directory contains Mercadia POS frontend workspaces, applications, shared UI packages, and
generated frontend API clients.

Expected applications:

- `apps/admin-web`
- `apps/pos-terminal`
- `apps/sco-terminal`
- `apps/senior-cashier-web`
- `apps/senior-cashier-terminal`
- `apps/assistant-station`

Expected future shared areas:

- `packages/ui`
- `packages/api-clients`
- `packages/design-tokens`
- `packages/testing`

Do not place backend services or backend packages here.
